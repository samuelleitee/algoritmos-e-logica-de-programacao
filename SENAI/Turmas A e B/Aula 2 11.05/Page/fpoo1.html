FPOO - Parte 2

Estruturas De Repetição

  • Existem os loops while e for, que são utilizados para automatizar tarefas repetitivas;
  • Vantagens: código menos repetitivo e mais prático para corrigir;
  • O while utiliza um contador antes de iniciar o loop para definir o seu fim;
  • O for itera sobre uma sequência definida.

Listas

  • Listas são estruturas de dados utilizadas para adicionar vários elementos dentro de uma única variável;
  • Uma lista fica dentro de um par de colchetes ([ ]) e os seus elementos são separados por vírgulas;
  • Em uma lista, é possível acessar um elemento específico através do índice, acrescentar um ou mais elementos, ordenar, realizar operações, e muito mais;
  • Existem outras estruturas de dados, como arrays, tuplas e dicionários.

Matrizes/Arrays

  • O Pyhton não tem suporte nativo para matrizes, mas é possível utilizá-las instalando a biblioteca NumPy através do comando pip instal numpy;
  • Com arrays, é possível armazenar dados de forma mais complexa, já que é possível criar um número ilimitado de linhas.

Funções

  • Funções são partes de um código que só são executadas, caso sejam chamadas;
  • No Pyhton, uma função é criada através da palavra-chave def;
  • Para chamar uma função, é necessário digitar o nome da função, seguido de um par de parêntesis (( )).

Orientação a Objetos

  • O Python, assim como várias outras linguagens, é orientado a objetos;
  • Na orientação a objetos, existem classes, objetos, métodos, atributos, entre outras coisas;
  • Uma classe é criada através da palavra-chave class, e ela pode conter atributos (características) e métodos(ações);
  • Um objeto é uma variável que chama uma classe para realiza determinada ação.
